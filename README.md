> notice
	
	为了避免servlet和struts2的路径撞车  
	
	servlet的路径命名逻辑改为:/object.servlet 
	
	例如AlbumServlet的路径,由原来的/album改为 
	
	/album.servlet 
	
	/album经过action类,访问效果和原来一样
	
	具体可以到
	
	web.xml,com.bdqn.qqmusic.action.AlbumAction,com.bdqn.qqmusic.servlet.AlbumServlet
