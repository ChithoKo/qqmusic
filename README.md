pull request 就是申请合并到原来的项目里

branch就是分支 就是说不同版本 一个项目比如我们做的新闻

可能最开始你只用jsp做 这是一个版本 后来改成用servlet 可以开一个新的branch

然后经过一些讨论啊 检查啊 什么鬼的 就可以合并 就是merge
